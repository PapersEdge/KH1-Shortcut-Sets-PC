# KH1-Shortcut-Sets-PC
This lua mod gives you 3 pages of Shortcuts instead of only 1 for a grand total of 9 Shortcuts!
You can edit the Shortcuts by going to the Customize Menu in game and using the DPad Left and Right buttons to scroll through the 3 pages, then add or remove Shortcuts like normal. Gone are the days of having to go into the Magic menu like a caveman!

To use the shortcuts, hold down L1 like usual, and after you're done admiring the pretty colors of the new Shortcut Page 1, use DPad Left and Right to cycle through the pages and use ALL your magic commands.
